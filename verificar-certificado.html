¿Qué significa esto? El código del certificado es un hash criptográfico SHA-256 del nombre del titular, el curso y la fecha de emisión. Al recomputar el hash en tu navegador y compararlo con el código, comprobamos que el contenido no ha sido alterado. La validación se hace en tu equipo — ningún dato se envía a un servidor.

🎓 ¿Cómo se verifica un certificado de Aziendale?
  1. El estudiante genera el certificado al completar el curso. El sistema produce un código único tipo EXO-IVA-2026-A3F7B2C1.
  2. El código es un hash SHA-256 derivado de forma determinística del nombre, el curso y la fecha — no se puede falsificar sin recalcular el hash.
  3. El estudiante comparte el enlace completo del certificado (con nombre, curso, fecha y código en el URL).
  4. Quien recibe el enlace abre esta página, el navegador recomputa el hash en local y lo compara con el código. Si coinciden, el certificado es auténtico.
